# Portfolio — Nicolas (Développeur Front-End)

Bienvenue sur mon portfolio personnel.

L’objectif de ce site est de présenter mon travail, mes compétences et mes projets à travers une interface animée et immersive.

---

## Fonctionnalités

### Animations GSAP custom

- Déformations dynamiques du texte en fonction de la vitesse du scroll
- SplitText + CustomBounce pour des effets typographiques
- Parallax

### Expérience utilisateur

- Effet de hover avancé
- Hero dynamique avec motion-inertia
- Apparition progressive des contenus

### Carousel Draggable

- Navigation horizontale sans scroll
- Contrôles précis avec GSAP Draggable

## Technologies

- **React**
- **SCSS**
- **GSAP**
